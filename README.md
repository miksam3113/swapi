## Installation

```bash
$ npm install
```

## Migration Database

```bash
# create tables
$ npm run migration:run

# delete created tables
$ npm run migration:down
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
